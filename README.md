# VoomBot Simulator

## Requirements

* Python >= 3.6
* pytest (only for unittests)

## Input format

Examle:

    5 5
    1 2 N
    LMLMLMLMM
    3 3 E
    MMRMMRMRRM
    ...

## Run

For easy testing, VoomBot Simulator can read from _stdin_ and output to _stdout_.

    cat test | python voombot_simulator.py
